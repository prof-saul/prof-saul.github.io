Introducción

¿Qué es SQL?

SQL (Structured Query Language o Lenguaje de Consulta Estructurada) es el lenguaje estándar utilizado en todo el mundo para comunicarse con bases de datos relacionales.

¿Para qué sirve?

Permite realizar operaciones clave sobre la información de una empresa o aplicación:

  • ✓ Consultar y buscar datos
  • ✓ Insertar nuevos registros
  • ✓ Actualizar información
  • ✓ Eliminar registros

Sintaxis Básica de Consulta (SELECT)

Para pedirle información a una tabla, usamos la estructura fundamental de consultas:

SELECT ¿Qué columnas quieres traer?

Usa * para traer todas las columnas, o escribe sus nombres separados por comas (ej. titulo, autor).

FROM ¿De qué tabla provienen los datos?

En este caso, nuestra tabla siempre será Libros.

WHERE ¿Bajo qué filtro o condición? (Opcional)

Filtra filas específicas. Ej. categoria = 'Literatura'. Las cadenas de texto se colocan siempre entre comillas simples '...'.

ORDER BY ¿Cómo quieres ordenar los resultados? (Opcional)

Ordena los resultados basándose en una columna de forma ascendente (ASC, por defecto) o descendente (DESC).

-- Ejemplo: Libros de aventura ordenados
SELECT *
FROM Libros
WHERE categoria = 'Aventura'
ORDER BY titulo ASC;
