🤔 ¿Qué es un ciclo?

Un ciclo (o bucle) es una instrucción que le dice a la computadora: "repite esto varias veces".

💡 Ejemplo de la vida real

Imagina que un maestro dice: "Cada alumno pase al frente a presentarse". ¡Eso es un ciclo! Se repite la misma acción para cada alumno.

¿Cuántas veces quieres aplaudir? 👏

3

La computadora repite "👏" el número de veces que elijas

📋 Tipos de ciclos que aprenderás

🔢
FOR — Cuando sabes cuántas veces repetir
🔁
WHILE — Mientras una condición se cumpla
📦
FOR EACH — Para cada elemento de una lista

🔢 Ciclo FOR

Úsalo cuando sabes exactamente cuántas veces quieres repetir algo.

🏃 Analogía

Es como decir: "Da 5 vueltas a la cancha". Sabes desde el inicio cuántas son.

📝 SEUDOCÓDIGO

// Desde 1 hasta 5:
PARA i = 1 HASTA 5 HACER:
  MOSTRAR ⭐
FIN PARA

💻 CÓDIGO REAL (JavaScript)

// Imprimir estrellas
for (let i = 1; i <= 5; i++) {
  mostrar("⭐");
}
5

🧩 ¿Cómo funciona?

1Inicio: let i = 1 → empieza en 1
2Condición: i <= 5 → ¿todavía es menor o igual?
3Acción: Muestra una estrella ⭐
4Incremento: i++ → suma 1 y vuelve al paso 2

🔁 Ciclo WHILE

Úsalo cuando no sabes cuántas veces se repetirá, solo sabes la condición para detenerse.

🍕 Analogía

"Mientras tengas hambre, come otra rebanada de pizza". No sabes cuántas comerás, ¡depende de tu hambre!

📝 SEUDOCÓDIGO

MIENTRAS hambre > 0 HACER:
  COMER 🍕
  hambre = hambre - 1
FIN MIENTRAS

💻 CÓDIGO REAL (JavaScript)

// Comer mientras haya hambre
let hambre = 5;

while (hambre > 0) {
  comer("🍕");
  hambre = hambre - 1;
}
5

⚠️ ¡Cuidado con el ciclo infinito!

Si la condición nunca se vuelve falsa, el ciclo nunca para. Es como decir "mientras estés vivo, respira" — ¡nunca termina!

// ❌ ¡CICLO INFINITO! Nunca para
while (true) {
  mostrar("¡Auxilio!");
}

📦 Ciclo FOR EACH

Úsalo para recorrer cada elemento de una lista, uno por uno.

📚 Analogía

"Para cada libro en tu mochila, ponlo sobre la mesa". Repites la acción con cada objeto de la lista.

📝 SEUDOCÓDIGO

// Lista de frutas
frutas = [🍎, 🍌, 🍇, 🍊, 🍓]

PARA CADA fruta EN frutas HACER:
  MOSTRAR fruta
FIN PARA

💻 CÓDIGO REAL (JavaScript)

let frutas = ["🍎", "🍌", "🍇", "🍊", "🍓"];

frutas.forEach(function(fruta) {
  mostrar(fruta);
});

🔄 FOR vs FOR EACH

FOR: Tú controlas el contador (i = 0, 1, 2...)
FOR EACH: Automáticamente toma cada elemento, sin necesidad de contador
