¿Cómo funciona la Web?

Imagina que estás construyendo una casa. Necesitas planos, decoración y electricidad.

HTML: El Esqueleto

Es el contenido. Define qué hay en la página: títulos, párrafos e imágenes.

<h1>Hola Mundo</h1>
<p>Esta es mi web</p>

CSS: El Estilo

Es el diseño. Define los colores, tamaños, fuentes y cómo se acomodan las piezas.

h1 { color: blue; }
p { font-size: 20px; }

JS: El Cerebro

Es la interactividad. Permite que pasen cosas cuando haces clic o escribes algo.

function saludar() {
  alert("¡Hola!");
}
